This file shows the code to produce figure 1, figure 2, and figure 3 in the main part of the paper.

The scripts should run in the following order:

1. BC1.R; GMM clustering and visualisation, export V for VB-GMM in Python script.
2. BRCA1.ipynb; VB-GMM clustering
3. BC1.R; import the posterior obtained from step 2 to visualise VB-GMM clustering results.
